# T-0.1.2: Configurar go.mod con team-ai-toolkit

**Historia:** HU-0.1 — Setup del proyecto
**Tipo:** Config
**Estado:** ⬜ Pendiente

---

## Descripción

Inicializar `go.mod` con el módulo `github.com/educabot/alizia-api` y agregar `team-ai-toolkit` como dependencia.

## Dependencias Go principales

```
github.com/educabot/team-ai-toolkit   → web, boot, tokens, dbconn, errors, pagination, applog, config
gorm.io/gorm                          → ORM (estándar empresa)
gorm.io/driver/postgres               → Driver PostgreSQL para GORM
github.com/stretchr/testify           → Testing assertions
```

## Pasos

1. `go mod init github.com/educabot/alizia-api`
2. `go get github.com/educabot/team-ai-toolkit`
3. `go get gorm.io/gorm gorm.io/driver/postgres`
4. `go get github.com/stretchr/testify`
5. Verificar que compila: `go build ./...`

## Notas

- team-ai-toolkit debe estar taggeado (v0.1.0 o similar) para que go mod lo resuelva
- Si el repo es privado, configurar `GOPRIVATE=github.com/educabot/*`
