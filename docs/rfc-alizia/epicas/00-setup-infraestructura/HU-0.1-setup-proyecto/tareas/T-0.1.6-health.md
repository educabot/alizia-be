# T-0.1.6: Endpoint /health

**Historia:** HU-0.1 — Setup del proyecto
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Implementar el endpoint GET `/health` usando `team-ai-toolkit/boot` que ya lo incluye por defecto en `NewEngine()`.

## Comportamiento

```
GET /health
→ 200 {"status": "ok"}
```

## Implementación

El `boot.NewEngine()` de team-ai-toolkit ya registra `/health` automáticamente. Solo hace falta:

1. Crear `cmd/main.go` con `boot.NewServer()` + `boot.NewEngine()`
2. Crear `config/config.go` embebiendo `config.BaseConfig`
3. Verificar que arranca y responde

## Archivos

- `cmd/main.go` — Entrypoint: crea engine, server, y arranca
- `config/config.go` — Config struct con `BaseConfig` embedded
