# T-1.1.4: Config: Auth0 env vars

**Historia:** HU-1.1 — Autenticación con Auth0
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Agregar las variables de configuración de Auth0 al config struct de la app.

## Implementación

```go
// config/config.go
type Config struct {
    config.BaseConfig // de team-ai-toolkit

    Auth0Domain   string
    Auth0Audience string
}

func Load() Config {
    base := config.LoadBase()
    return Config{
        BaseConfig:    base,
        Auth0Domain:   config.MustEnv("AUTH0_DOMAIN"),
        Auth0Audience: config.MustEnv("AUTH0_AUDIENCE"),
    }
}
```

## Variables de entorno

| Variable | Descripción | Ejemplo |
|----------|-------------|---------|
| `AUTH0_DOMAIN` | Auth0 tenant domain | `educabot-staging.auth0.com` |
| `AUTH0_AUDIENCE` | Auth0 API identifier | `https://api.alizia.educabot.com` |

## Notas

- `MustEnv` panics si la variable no existe — fail fast al startup
- `BaseConfig` ya trae: `PORT`, `DATABASE_URL`, `ENV`, `BUGSNAG_API_KEY`
