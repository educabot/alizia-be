# T-1.4.2: Entities y providers para asignaciones

**Historia:** HU-1.4 — Asignación organizacional
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Definir las entities Go y las interfaces de provider para las tablas de asignación organizacional.

## Entities

```go
// internal/admin/entities/area_coordinator.go
type AreaCoordinator struct {
    ID        uint      `gorm:"primaryKey"`
    AreaID    uint      `gorm:"not null"`
    UserID    uint      `gorm:"not null"`
    CreatedAt time.Time

    // Relations
    Area *Area `gorm:"foreignKey:AreaID"`
    User *User `gorm:"foreignKey:UserID"`
}

// internal/admin/entities/course_subject.go
type CourseSubject struct {
    ID             uint      `gorm:"primaryKey"`
    CourseID       uint      `gorm:"not null"`
    SubjectID      uint      `gorm:"not null"`
    TeacherID      uint      `gorm:"not null"`
    OrganizationID uint      `gorm:"not null"`
    SchoolYear     int       `gorm:"not null"`
    CreatedAt      time.Time

    // Relations
    Course  *Course  `gorm:"foreignKey:CourseID"`
    Subject *Subject `gorm:"foreignKey:SubjectID"`
    Teacher *User    `gorm:"foreignKey:TeacherID"`
}
```

## Providers

```go
// internal/admin/providers/area_coordinator_repository.go
type AreaCoordinatorRepository interface {
    Assign(ctx context.Context, areaID, userID uint) (*entities.AreaCoordinator, error)
    Remove(ctx context.Context, areaID, userID uint) error
    FindByAreaID(ctx context.Context, areaID uint) ([]entities.AreaCoordinator, error)
    FindByUserID(ctx context.Context, userID uint) ([]entities.AreaCoordinator, error)
    IsCoordinator(ctx context.Context, areaID, userID uint) (bool, error)
}

// internal/admin/providers/course_subject_repository.go
type CourseSubjectRepository interface {
    Create(ctx context.Context, cs *entities.CourseSubject) error
    FindByCourseID(ctx context.Context, courseID uint) ([]entities.CourseSubject, error)
    FindByTeacherID(ctx context.Context, teacherID uint, orgID uint) ([]entities.CourseSubject, error)
    FindByID(ctx context.Context, id uint) (*entities.CourseSubject, error)
}
```

## Notas

- `IsCoordinator` es un helper para validar rápido si un usuario coordina un área (usado en authz de documentos)
- `FindByTeacherID` siempre filtra por `organization_id` (multi-tenancy)
- Las relaciones de GORM permiten hacer `Preload("Teacher")`, `Preload("Subject")` etc.
